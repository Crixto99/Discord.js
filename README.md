# **Discord Bot**

### Dependencias utilizadas
* **Discord.js**
* **Dotenv**
* **Fs** 
* **Path**
----

### Debes colocar en tu terminal:

```javascript
npm i discord.js dotenv fs mongoose path && npm i -D nodemon
```

### Ir a tu package.json y colocar abajo de scripts:
--------------

```javascript
"start": "nodemon ./src/index.js",
"dev": "node ."
```
